Jan-Oliver Pantel - SOLID - 5 characters for better code

As everything object-oriented software has its benefits and pitfalls.

When it comes to good code design the SOLID pattern is the way to go. With ES6 (and also today with frameworks like Angular) also frontend developers are empowered to write SOLID code. In this talk I will show you how you power up your code quality by sticking to those 5 simple rules.

Beside this we will have a look at some real world patterns that are related to SOLID.
