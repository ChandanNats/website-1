Christoph Burgdorf - ninya.io - a search engine to make skilled people in tech more discoverable

ninya.io is all about making skilled people in tech more discoverable. It might be best described as a search engine to find tech people. Basically it let's you search for people by their skillset and location. For instance here are some people from Hannover who seem to be web folks!

Christoph started ninya.io as a quick holiday hack in August 2013. Since then it has grown into a serious project with it's own set of challenges and fun parts.

In his talk Christoph will talk about the tech stack, including it's turbulent history, challenges and future plans.

As a bonus you might see some Go code in this talk!

Sven Ramuschkat - Amazon Web Services and how to use them for WebApps

Amazon Web Services is a rock solid foundation with a large basket ob building bricks for your web or mobile applications.

In this session you will get a general overview about the most important AWS Services and concepts and also some examples how to use AWS Services directly via the JavaScript API (including adhoc Authentification via the AWS Security Token Services (STS).

Sven Ramuschkat from tecRacer is an authorized Amazon Web Services Trainer and has more than 3 years experience with AWS.

Jan-Oliver Pantel - Advanced data modeling in the frontend

In the days of modern frameworks (like Angular), handling data becomes more and more important to frontend devs. Just using simple AJAX calls and working with dumb JSON object became a pain if your application gets more complex.

I came accross this problem when starting a side project, for which I've used Angular in the frontend and a Laravel PHP based REST-API for the backend. So I decided to implement a frontend ORM which became pretty powerful.

Since you can't generalize an ORM for a frontend because every backend has it's own rules, I will give you a starting point how you can approach an own ORM implementation for you upcomming projects.
