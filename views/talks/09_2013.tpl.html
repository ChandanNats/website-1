Pascal Precht — Hands on Polymer

Polymer is a new type of library for the web, built on top of WebComponents, and designed to leverage the evolving web platform on modern browsers.

Pascal will give you a round trip trough the Polymer world. And how it'll change the world of client side web applications.

Everything is a an element!

Christoph Burgdorf — 5 things you didn't know about rendering performance

Keeping a web app to run smoothly on 60fps can be a challenging task. Lot's of things can influence performance and are often to be found at quite some hidden places.

Christoph collected a bunch of lesser known things that have a performance impact which you should be aware of.
