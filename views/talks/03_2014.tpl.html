Talks

Pascal Precht - Taking angular-translate to the next level

Bringing i18n to Angular apps works like a charm when using the angular-translate module. We've been working on version 2.0 for months now and it brings plenty new features.

In this talk, i'll show you the new features and breaking changes that come with angular-translate 2.0.

Jan-Oliver Pantel - Amazing realtime apps with sails and Angular

When it comes to realtime apps, there are plenty of cool frameworks on the line. Sails introduces a node based MVC backend framework. With its awesome socket.io routing and fully RESTish design developers will love to work with it.

This talk will cover a description of sails and its functionalities. Also the connection to frontend frameworks (in this case Angular) will be presented.

Christoph Burgdorf - sofa.io - the CouchCommerce Web App SDK for Magento, Shopware, OXID and more

Traditional e-commerce systems allow to use responsive templates only. The one who likes to impress his customers with a real slick Web App that feels like a native app is still left alone in the rain.

CouchCommerce has built an SaaS platform which links up to the customers traditional shop system and generates a Web App as an alternative frontend. The technology behind that is now open sourced as the sofa project. Together with the CouchCommerce platform and the sofa project developers and agencies can create e-commerce Web Apps for lots of different shopping systems in a much easier and more efficient way.

In his talk Christoph will show how to build a simple demo shopping app with sofa and AngularJS
