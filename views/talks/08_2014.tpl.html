Robin Thrift - Metalsmith - A fun, pluggable JavaScript static site generator

Imagine Jekyll mixed with Gulp and a little bit of magic. Add to that a dead simple plugin system and you've got yourself a pretty darn amazing static site generator. That's what the folks over at Segment.io thought to themselves and built Metalsmith, a pluggable JavaScript static site generator.

This talk will show you, exactly how easy it is, to build static websites using Metalsmith. From the first parsed markdown file to building a super simple plugin, you'll fall in love with Metalsmiths simplicity and its declarative nature.
