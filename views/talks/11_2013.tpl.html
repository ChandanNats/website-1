Talks

Emanuel Kluge — Modular JavaScript

One important aspect of writing software is to keep the code modular. The goal is to have a bundle of distinct pieces, that can be combined at pleasure. Each module ideally serves one purpose.

Emanuel will give you a brief introduction into the topic. After that he'll shows how to achieve these goals with either RequireJS or Browserify — two interesting module systems.

Felix Wahner — Javascript ALL THE THINGS!

With upcoming controllers like Espruino and Tessel as well as new, programmable input devices (leap motion, ocolus rift) and their respective JS-APIs, javascript is steadily evolving and shifting into the physical world.

Felix will give a short introduction about the current state of "javascript in the real world" and bring some ideas and stuff to play with.
