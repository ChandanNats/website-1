The Talks

Snabbdom: A lightweight virtual DOM library

Karol Wegner

Freelance full-stack developer

Snabbdom is a lightweight virtual DOM library which is used in frameworks like Vue.js and Cycle.js as the DOM renderer.

But with under 4KB size (gzipped+minified) and a highly modular structure, Snabbdom is also very useful for building small (and not so small) dynamic web UI components without committing to a framework.

The talk will demonstrate how Snabbdom works and how to build simple UI components with it "functional style".

What Is This All About?
